# Jitsi-meet-Zimlet

This zimlet allows you to add a Jitsi Meet URL your message body (email and appointment)  

##### Install the Jitsi-meet-Zimlet
To install the Jitsi-meet-Zimlet you can follow theses steps :  
    git clone the project  
    chmod +x install.sh  
    ./install.sh
